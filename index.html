Sobre mim

Olá! Seja bem vindo ao meu Portfólio

Me chamo Vinícius Griebler estudante de Análise e desenvolvimento de Sistemas e desenvolvedor Full Stack em formação!

Iniciei minha carreira profissional na indústria e atualmente atuo em uma empresa de tecnologia, como Analista de Sistema MES (Manufaturing Execution System) responsável pela coleta e análise de dados de produção. Paralelamente, mantenho minha paixão por programação e desenvolvimento de software, com grande interesse em criar soluções eficientes e inovadoras. Acredito que minhas habilidades técnicas, aliadas à experiência prática, podem contribuir de forma significativa para a resolução de problemas e para o crescimento de equipes de desenvolvimento.

Além da minha atuação profissional, valorizo atividades que contribuem para o meu equilíbrio pessoal. Gosto de viajar, aproveitar o clima praiano, estar em contato com familiares e amigos, também aprecio muito maratona de séries e filmes. Além disso, busco me manter constantemente atualizado sobre as novidades e tendências do mundo da tecnologia.

Estou sempre me desafiando com novos projetos e participando de comunidades de programação para buscar feedback de outros desenvolvedores e desenvolvedoras. Além de tentar ajudar essas pessoas com o que eu já aprendi.

Habilidades

HTML5

Minhas habilidades avançadas em HTML me permitem aplicar as melhores práticas de marcação para construir estruturas sólidas, organizadas e semânticas, desenvolvendo páginas bem estruturadas, acessíveis e responsivas, sempre priorizando boas práticas de SEO, legibilidade e manutenção do código..

CSS3

Meu conhecimento avançado em CSS me ajuda a aplicar técnicas modernas para estilizar páginas, sites e sistemas de forma visualmente atraente e coerente com paletas de cores bem definidas e modernas. Desenvolvo interfaces responsivas, assegurando uma experiência consistente em dispositivos mobile, tablet e desktop, utilizando media queries e boas práticas de design responsivo.

TypeScript

Possuo conhecimento intermediário em TypeScript, desenvolvendo aplicações tipadas, a prova de falhas no código, organizadas e escaláveis com integração em projetos utilizando React, Next.js, Node.js e também Banco de dados.

SQL Server

Atualmente trabalho bastante com Microsoft SQL Server, realizando consultas, manipulação e análise de dados utilizando SQL na resolução de problemas. Possuo experiência com criação de queries, procedures, CRUD, filtros, ordenações e validações em banco de dados, sempre buscando organização, desempenho e integridade das informações.

PostgreSQL

Possuo conhecimento acadêmico em PostgreSQL, com boa compreensão de modelagem e manipulação de bancos de dados relacionais. Tenho facilidade para desenvolver queries, realizar operações CRUD, criar relacionamentos entre tabelas e trabalhar com consultas e organização de dados.

React.js

Estou em fase de aprendizado em React, desenvolvendo componentes reutilizáveis para páginas web. Tenho praticado a criação de interfaces utilizando conceitos como componentização, props e organização de código, buscando evoluir continuamente no desenvolvimento de aplicações front-end.

Next.js

Estou me especializando em Next.js, aprofundando meus conhecimentos no desenvolvimento de aplicações modernas, performáticas e escaláveis. Já possuo projetos com Next, bom conhecimento de integração com TypeScript, Node.js e SQL, aplicando conceitos de componentização, rotas, consumo de APIs e organização de código para construir aplicações completas e eficientes.

Prisma

Tenho conhecimento em Prisma ORM para integração entre aplicações e banco de dados, realizando modelagem, consultas e manipulação de dados com TypeScript de forma prática, organizada e eficiente.

Git

Utilizo também muito o Git como ferramenta de versionamento de código, garantindo o controle eficiente de alterações e a organização de projetos. Minha experiência inclui colaboração em equipe, utilizando boas práticas de branch, merge e commit, o que permite desenvolvimento conjunto de forma estruturada, rastreabilidade das mudanças e maior qualidade no código entregue.
