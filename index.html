Olá! Seja bem vindo ao meu Portfólio

Me chamo Vinícius Griebler estudante de Análise e desenvolvimento de Sistemas e desenvolvedor Front-End em formação!

Iniciei minha carreira profissional na indústria e atualmente atuo em uma empresa de tecnologia, prestando suporte a um Sistema MES (Manufaturing Execution System) responsável pela coleta e análise de dados de produção das máquinas. Paralelamente, mantenho minha paixão por programação e desenvolvimento de software, com grande interesse em criar soluções eficientes e inovadoras. Acredito que minhas habilidades técnicas, aliadas à experiência prática, podem contribuir de forma significativa para a resolução de problemas e para o crescimento de equipes de desenvolvimento.

Além da minha atuação profissional, valorizo atividades que contribuem para o meu equilíbrio pessoal. Gosto de viajar, aproveitar o clima praiano, estar em contato com familiares e amigos, também aprecio muito maratona de séries e filmes. Além disso, busco me manter constantemente atualizado sobre as novidades e tendências do mundo da tecnologia.

Estou sempre me desafiando com novos projetos e participando de comunidades de programação para buscar feedback de outros desenvolvedores e desenvolvedoras. Além de tentar ajudar essas pessoas com o que eu já aprendi.

Habilidades

JavaScript

Possuo conhecimento básico em JavaScript, com experiência na criação de scripts voltados para manipulação de elementos do navegador e interação com sistemas e sites. Já desenvolvi funcionalidades que envolvem manipulação do DOM, eventos, ajustes dinâmicos de interface e automação simples de comportamentos no navegador.

HTML5

Tenho habilidades avançadas em HTML, aplicando as melhores práticas de marcação para construir estruturas sólidas, organizadas e semanticamente corretas. Desenvolvo o "esqueleto" das páginas de forma eficiente, priorizando acessibilidade, SEO e padrões modernos da web. Minha experiência inclui a criação de layouts bem estruturados, utilizando tags semânticas apropriadas para garantir clareza, desempenho e fácil manutenção do código.!

CSS3

Meu conhecimento avançado em CSS me ajuda a aplicar técnicas modernas para estilizar páginas, sites e sistemas de forma visualmente atraente e coerente com paletas de cores bem definidas. Desenvolvo interfaces responsivas, assegurando uma experiência consistente em dispositivos mobile, tablet e desktop, utilizando media queries e boas práticas de design responsivo. Também faço uso de propriedades avançadas, como flexbox e grid, garantindo layouts organizados, fluidos e de fácil manutenção.

GitHub

Faço o uso do GitHub como plataforma principal para versionamento e publicação dos meus projetos. Por meio dele, organizo meu código, acompanho o histórico de alterações e mantenho meus repositórios atualizados. Além disso, utilizo boas práticas de commit e estruturação de projetos, facilitando colaboração, documentação e acompanhamento da evolução das minhas habilidades de desenvolvimento.

Git

Utilizo também muito o Git como ferramenta de versionamento de código, garantindo o controle eficiente de alterações e a organização de projetos. Minha experiência inclui colaboração em equipe, utilizando boas práticas de branch, merge e commit, o que permite desenvolvimento conjunto de forma estruturada, rastreabilidade das mudanças e maior qualidade no código entregue.

React

Estou em fase de aprendizado em React, desenvolvendo componentes reutilizáveis para páginas web. Tenho praticado a criação de interfaces utilizando conceitos como componentização, props e organização de código, buscando evoluir continuamente no desenvolvimento de aplicações front-end.
